BoseHubbardGutzwiller

This code implements the homogeneous Gutzwiller variational wave function for the Bose-Hubbard model. The search for the optimal wave-function parameters is performed through Simulated Annealing, a Monte Carlo method for stochastic optimization.
Links to code: Zenodo, GitHub
Author: Tommaso Comparin
DOI identifier: 10.5281/zenodo.846904
Used in: Chaviguri et al. [Phys. Rev. A 97, 023614 (2018)], Huembeli et al. [Phys. Rev. B 97, 134109 (2018)].

Laughlin-Metropolis

This code implements the Metropolis Monte Carlo algorithm to sample configurations from the Laughlin wave function with an arbitrary number of quasiholes (see for instance Morf&Halperin, 1986).
Links to code: Zenodo, GitHub
Authors: Tommaso Comparin, Elia Macaluso
DOI identifier: 10.5281/zenodo.1161969
Used in: Umucalilar et al. [arXiv:1712.07940]
